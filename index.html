Solving Techniques

Naked Single Beginner
A cell has only one candidate remaining. All other numbers are already present in its row, column, or box. This is the most basic technique — simply fill in the only possibility.

Hidden Single Beginner
Within a row, column, or box, a number can only go in one cell. Even though that cell has multiple candidates, this number has no other valid position in that unit.
Used: 0 times
Naked Pair Intermediate
Two cells in the same unit share the exact same two candidates. These two numbers must go in these two cells, so they can be eliminated from all other cells in that unit.
Used: 0 times
Hidden Pair Intermediate
Two numbers appear as candidates in only the same two cells within a unit. Other candidates in those two cells can be safely removed, leaving only the pair.
Used: 0 times
Pointing Pair Intermediate
Within a box, a candidate is restricted to a single row or column. That number must go somewhere in that line within the box, so it can be eliminated from the rest of that row/column outside the box.
Used: 0 times
Box/Line Reduction Intermediate
Within a row or column, a candidate is confined to a single box. Since that number must appear in that line within the box, it can be eliminated from other cells in the same box.
Used: 0 times
Naked Triple Intermediate
Three cells in a unit collectively contain only three candidates. Those three numbers are locked to these cells and can be eliminated from all other cells in the unit.
Used: 0 times
Hidden Triple Advanced
Three numbers appear as candidates in only three cells within a unit. All other candidates in those three cells can be removed.
Used: 0 times
X-Wing Advanced
A candidate appears in exactly two cells in each of two rows, and those cells align in the same columns. The number must be in opposite corners, so it can be eliminated from other cells in those columns. Works symmetrically with columns/rows.
Used: 0 times
XY-Wing Advanced
Three cells form a chain: pivot(XY), wing1(XZ), wing2(YZ). The pivot sees both wings. No matter which value the pivot takes, Z is eliminated from cells that see both wings.
Used: 0 times
Swordfish Expert
Extension of X-Wing to three rows and three columns. A candidate appears 2-3 times in each of three rows, all within the same three columns. Eliminates that candidate from other cells in those columns.
Used: 0 times
Bifurcation Expert
When all logical techniques are exhausted, guess a value for the cell with fewest candidates. If a contradiction is found, backtrack and try the next option. Humans call this "what if" or trial-and-error.
